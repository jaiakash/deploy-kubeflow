# Deploy Kubeflow on OCI

A streamlined Terraform automation for deploying Kubeflow on Oracle Cloud Infrastructure (OCI).

## 🚀 Quick Start

1. **OCI Auth**: Setup keys and OCIDs in [OCI Authentication](guides/OCI_AUTH.md).
2. **Deploy Cluster**: Provision OKE in [Cluster Deployment](guides/CLUSTER_DEPLOY.md).
3. **Install Kubeflow**: Setup the platform in [Kubeflow Installation](guides/KUBEFLOW_INSTALL.md).

## 📚 Documentation

- [1. OCI Setup & Auth](guides/OCI_AUTH.md)
- [2. Cluster Deployment](guides/CLUSTER_DEPLOY.md)
- [3. Kubeflow Installation](guides/KUBEFLOW_INSTALL.md)
- [Troubleshooting](guides/TROUBLESHOOTING.md)

## 🛠 Prerequisites

- OCI Account with OKE Quotas.
- Terraform >= 1.5.0.
- OCI CLI installed and configured.
